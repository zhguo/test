## Rate 

评分。

## 使用

```js
<o-rate 
  total={5} 
  value={this.valueA} 
  color='#07C160' 
  onSelect={this.onSelect} />
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------|
| total         |   number  |    --   |      总的星星个数       |　
| value         |    number  |    --   |   支持小数打半星          |　
| color         |   string  |    --   |             |　
| onSelect         |    function  |    --   |             |
