## Progress 

展示一个任务的进度.

## 使用

```js
<o-progress bgColor="#ccc" innerColor="#07C160" value={50} />
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| value         | number|    0   |      最小是0，最大是100       |
| bgColor         | string|    #CCCCCC   |        背景色     |
| innerColor         | string|    #07C160   |     进度条的颜色        |
