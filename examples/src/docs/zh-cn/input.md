## Input  

用户的输入框，输入一些内容用来提交给服务器。

## 使用

```js
<o-input  placeholder="请输入用户名" />
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| placeholder  | string|       |     占位字符串      |

## 其他类型

```js
<o-input-label label='UserName' />
<o-input-button buttonText='提交' onClick={this.onClick} />
```
