## InputNumber 

可以调整的数字输入框。

## 使用

```js
<o-input-number
  onChange={this.onInputNumberChange}
  min={1}
  max={10}
  step={1}
  value={2}
  label="描述文字"
/>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| min  | number|--  ||
| max  | number| -- |  |
| step | number   |    | |
| value | number| -- ||
| label | string| -- ||
| onChange | function| -- ||
