## Badge 

图标右上角的圆形徽标数字。

## 使用

```js
<o-badge content={8}>
  <o-icon type="chat" isFill color="#07C160" />
</o-badge>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------|
| content         |   string  |    --   |             |　
| children         |    json array  |    --   |   数组的长度为 1          |
