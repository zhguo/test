## Radio  

单选框。

## Usage

```js
<o-radio name='group' checked label="my label" />
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| checked  | bool|   false    |           |
| label  | string|       |           |
| name  | string|       |           |
