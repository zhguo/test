## SelectList

选项列表。

## 使用

```js
<o-select-list
  onSelect={this.onSelect}
  list={this.list}
  selectedIndex={this.selectedIndex}
/>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| list  | json array|       |           |
| selectedIndex  | number |       |           |
| onSelect  | function|       |           |
