## Button 按钮 

点击或触摸触发一个操作的元素。响应用户点击操作，触发封装的逻辑。

## 使用

```js
<o-button>我是按钮</o-button>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| type  | string| primary |Options: primary, default, warn, vcode|
| size | string   |   normal |Options: normal, small|
| disabled | bool| false ||
