## Step  

工作流或步骤进度。

## 使用

```js
<o-step items={[
    { name: 'Finished', description: 'This is a description.', state: state.DONE },
    { name: 'In Progress', description: 'This is a description.', state: state.DOING },
    { name: 'Waiting', description: 'This is a description.', state: state.TODO }
  ]} />
```
