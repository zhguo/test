## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| paths  | json array |--  | pass in paths, path does not need to pass in  |
| path  | string| -- ||
| scale | number   |   2 | |
| color | string| black ||
| rotate | bool| false ||
| isFill | bool| false |deprecated|
| type  | string| -- |deprecated|
