## Rate 

Rate.

## Usage

```js
<o-rate 
  total={5} 
  value={this.valueA} 
  color='#07C160' 
  onSelect={this.onSelect} />
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------|
| total         |   number  |    --   |      Total number of stars       |　
| value         |    number  |    --   |   Support decimal, corresponding to half star          |　
| color         |   string  |    --   |             |　
| onSelect         |    function  |    --   |             |
